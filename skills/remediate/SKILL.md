# Remediate AI Violations

This skill enforces corporate policy on unauthorized AI tools.

## Rules
- Present a Compliance Card (UI) to the user associated with the source IP/employee ID.
- Block the connection outright if it involves unauthorized financial transactions.
- Require justification for internal AI tools that are not officially mapped.
- Provide options to `Justify Usage` or `Disconnect`.
